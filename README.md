## 🆕 What's in this release

### Core Features
- 🎯 Smart reply generation — reads tweet context before generating
- ✍️ Post idea generator — generates original tweets from your persona
- 🧵 Full thread generator — 6-tweet threads in one click
- ⚡ A/B tone tester — compare two tones side by side
- 👤 Persona profiles — multiple switchable personas

- 📋 Reply queue — save replies, mark as sent

- ↩ Undo insert — 4 second undo after inserting

## 🚀 Installation

1. Download `xreply-ai-extension.zip` below
2. Unzip it
3. Go to `chrome://extensions`
4. Enable **Developer Mode**
5. Click **Load unpacked** → select the `xreply-ai` folder
6. Get a free API key at https://console.groq.com
7. Paste it in the extension → Keys tab → Groq
